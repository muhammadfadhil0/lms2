🔑 Konfigurasi API:

• Page Name: "ai-evaluation-detail-jawaban"

• Lokasi Setting: Admin Settings → Manajemen Akses AI → Pengaturan Per Halaman

• Fallback: Otomatis menggunakan API key aktif pertama jika tidak dikonfigurasi

📋 Cara Penggunaan

👨‍💼 Untuk Admin:

  1. Buka Admin Settings → Manajemen Akses AI
  2. Tambah API Key baru atau gunakan yang sudah ada
  3. Di bagian "Pengaturan Per Halaman", cari "AI Evaluation - Detail Jawaban Siswa"
  4. Pilih API Key yang ingin digunakan untuk fitur AI evaluation
  5. Fitur siap digunakan oleh guru!

👩‍🏫 Untuk Guru:

  1. Masuk ke halaman Hasil Ujian
  2. Klik salah satu siswa untuk melihat Detail Jawaban
  3. Scroll ke section "Analisis Ujian Siswa dengan Pingo"
  4. Klik tombol "Mulai Analisis dengan AI"
  5. Tunggu AI menganalisis dan lihat hasil evaluasi lengkap!
